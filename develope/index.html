1. Standard Configuration

  • Name: BRS Token
  • Symbol: BRS
  • Decimals: 18
  • Initial supply: 1,000,000,000 BRS
  • Network: Ethereum mainnet
  • Token type: Customized ERC20
  • Solidity compiler: 0.8.2
  • Contract address: 0xFAd4fbc137B9C270AE2964D03b6d244D105e05A6

2. Network Extension

The Ethereum network is not suitable for use at the service level due to the problem of gas cost and the problem of block generation interval. BRS provides services by interchaining with other networks as needed.

1) Precautions when expanding the network

(1) Minting and Burning of Tokens

⚠️
Tokens issued by the extended network do not have a minting function.
Minting that affects the total supply of tokens is only possible with the main token based on the Ethereum network.

(2) Name of Tokens

  • The names of the tokens wrapped or mapped for each network are all BRS.
